## Software Requirements

- PHP >= 8.0
- Grav CMS >= 1.7.0 (included in the skeleton package)
- [Grav Premium Helios Theme](https://getgrav.org/premium/helios) – one license per site ([Standard or Team](https://getgrav.org/premium/license))

> [!NOTE]
> The Grav Premium Helios theme requires a paid license – one license per site. Purchasing the Helios theme directly supports the open-source Grav CMS project.

[definition title="Grav Premium Helios Theme"]
A commercial theme for Grav CMS that provides the reader layout, sidebar navigation, versioning support, and Admin Panel styling used by Helios Open Reader. Available at [getgrav.org/premium/helios](https://getgrav.org/premium/helios). Standard and Team license tiers are available.
[/definition]

## Pre-flight Checklist

Before starting the installation, confirm the following:

1. Your web server meets [Grav's requirements](https://learn.getgrav.org/17/basics/requirements) (PHP 8.0 or higher)
2. You have your web server login credentials ready (username and password)
3. You have purchased (or are ready to purchase) a Grav Premium Helios theme license
